Getting started

roblox-ts

  1. Install the module
  2. import trollsmile
  3. use it like a class 
    // default settings look a little like this
const trollsmile = new Trollsmile({
  devRank: false, // gives SunRaysEffect owner (pls enable lol)
  permission: 0, // the default permission level
  ranks: {},
  prefix: 't!',
  cmdOverrides: {}, // key is a command name, value is the permission number
  aliases: {},
  brand: 'trollsmile', // change this to change what it says in notifs and dialogs instead of trollsmile
  welcome: true
})
